Following are the instructions to install q_kernel

- copy 'q_kernel' directory to /usr/local/lib/python2.7/dist-packages/
$ cp -r Q/experimental/jupyter/q_kernel /usr/local/lib/python2.7/dist-packages/

- Install q_kernel
$ cd /usr/local/lib/python2.7/dist-packages/q_kernel
$ python install.py

- Validate q_kernel installation
$ jupyter kernelspec list
This would list the q kernel entry
